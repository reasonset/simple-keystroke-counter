# Simple Key Stroke Counter

## Main

### Description

This is a simple HTML page for measuring keystrokes per minute.

It was created to compare keyboard performance.

Please establish your own rules for how to conduct the measurement.

It also includes a display designed for rhythm games where rapid keystrokes are crucial.

### Usage

* Open `index.html`
* Just keep mashing the keys

### WWW version

## 3s/10s

### Description

Displays the number of keystrokes over the last 3 or 10 seconds.

Since it requires maintaining focus on the page, it is primarily used to measure instantaneous typing speed while viewing the problem text.
